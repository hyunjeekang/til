## 개요

* **목적:** 모든 정점 쌍 간의 최단 경로를 한 번에 구한다.
* **시간 복잡도:** $O(V^3)$ (노드 개수 $V$가 적을 때 유용)
* **조건:** 
    * 무향 그래프: 양수 가중치만 허용됨 (무향 그래프에서의 음수 간선은 그 자체로 음수 사이클이 됨!)
    * 유향 그래프: 양/음 가중치 모두 허용되나, 음수 사이클은 없어야 한다.
* **완전 탐색의 한계:** 모든 경로를 일일이 확인하는 것은 경로의 수가 $(V-2)!$ 가지에 달해 불가능<br><br>


## 플로이드-워셜 알고리즘<br>
거쳐가는 **'경유지'를 기준으로 문제를 쪼개**는 동적 계획법(DP)의 일종!<br>

### 1. 부분 문제 정의<br>
* $D(k, i, j)$: 1번부터 $k$번 노드까지만 중간 경유지로 사용 가능할 때, $i$에서 $j$로 가는 최단 거리

* $k$를 0에서부터 $N$까지 점진적으로 늘려가며 최적해를 구한다. k=N$일 때가 최종 정답!<br><br>

### 2. 점화식 도출<br>
새로운 경유지 $k$를 사용할 수 있게 되었을 때, 최단 경로는 다음 두 가지 경우 중 하나이다.
* $k$번 노드를 거치지 않는 경우: $D(k-1, i, j)$

* $k$번 노드를 거쳐 가는 경우: $D(k-1, i, k) + D(k-1, k, j)$

* **점화식:** $D(k, i, j) = \min \big( D(k-1, i, j), \; D(k-1, i, k) + D(k-1, k, j) \big)$<br><br>

### 3. 2차원 배열로의 공간 최적화<br><br>
점화식은 3차원이지만, 실제 구현 시에는 2차원 배열(`matrix[i][j]`)만을 사용한다. <br>
$k$번째 단계를 계산할 때 직전 단계($k-1$)의 값만 참조하며, 갱신 과정에서 $k$를 출발지나 도착지로 하는 값은 변하지 않기 때문에 배열을 덮어써도 논리적 오류가 발생하지 않는다<br><br>

### 4. 구현 코드
```python
for k in range(N):          # 1. 거쳐가는 중간 노드
    for i in range(N):      # 2. 출발 노드
        for j in range(N):  # 3. 도착 노드
            matrix[i][j] = min(matrix[i][j], matrix[i][k] + matrix[k][j])
```

## 추가 내용

### 밀집 그래프에서의 효율성 비교 

다익스트라 vs 플로이드-워셜

일반적으로 우선순위 큐(Heap)를 사용하는 다익스트라가 단순 배열을 사용하는 방식보다 빠르다고 알려져 있지만, 간선이 매우 많은 환경에서는 다를 수 있다<br><br>

**1. 그래프의 종류**
* **희소 그래프:** 정점 수 대비 간선 수가 적은 그래프

* **밀집 그래프:** 거의 모든 정점이 서로 연결된 그래프 (간선 수 $E \approx V^2$)<br><br>

**2. 다익스트라 구현 방식에 따른 시간 복잡도**
* 단순 1차원 배열: $O(V^2)$

* 우선순위 큐 (Heap) 사용: $O(E \log V)$<br><br>

**3. 밀집 그래프 ($E \approx V^2$) 대입 시**
* 단순 배열 다익스트라: $O(V^2)$

* **우선순위 큐 다익스트라:** $O(V^2 \log V)$
간선의 수가 압도적으로 많아지면 매번 간선을 힙에 넣고 재정렬하는 데 소요되는 비용($\log V$) 때문에 힙 구조의 오버헤드가 커져서, 단순 배열보다 오히려 처리 속도가 떨어질 수 있다.<br><br>

**4. 모든 정점 쌍 간의 최단 경로를 구할 때 ($V$번 반복)**
모든 정점에서 다익스트라 알고리즘을 실행하여 플로이드-워셜과 같은 결과를 얻으려 할 때의 시간 복잡도

* 단순 배열 다익스트라: $V \times O(V^2) = O(V^3)$

* 우선순위 큐 다익스트라: $V \times O(V^2 \log V) = O(V^3 \log V)$

* **플로이드-워셜:** $O(V^3)$

**결론:** 밀집 그래프에서는 구현이 훨씬 직관적이고 시간 복잡도 측면에서도 손해가 없는 플로이드-워셜 알고리즘($O(V^3)$)을 선택하는 것이 좋음!


### 최단 경로 알고리즘 분류
* **가중치 없는 그래프:** BFS(너비 우선 탐색)
* **가중치 있는 그래프:**
    * **양수 가중치만 존재:** 다익스트라(Dijkstra)
    * **음수 가중치 포함:** 벨만-포드(Bellman-Ford) 또는 플로이드-워셜(Floyd-Warshall)
